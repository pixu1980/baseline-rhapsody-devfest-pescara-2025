Scrollbar Styling

Newly Available (August 2024)

Customize scrollbars with standard CSS

Thin Scrollbar
This content scrolls with a thin custom scrollbar. Notice the purple thumb and light gray track. Scroll to see the effect!
Custom Colors
Green thumb with light green track. The scrollbar maintains the browser's default width while using custom colors.
Options
• thin - Minimal width
• auto - Platform default
• none - Hidden
• Custom thumb/track colors
 
        
/* Standard scrollbar styling */
.container {
  /* Width: thin, auto, none */
  scrollbar-width: thin;
  
  /* Colors: thumb track */
  scrollbar-color: #667eea #e5e7eb;
}

/* Thin scrollbar with custom colors */
.scrollable {
  scrollbar-width: thin;
  scrollbar-color: #10b981 #d1f4e8;
}

/* Hide scrollbar (still scrollable) */
.hidden-scroll {
  scrollbar-width: none;
}

/* WebKit specific (more control) */
.custom::-webkit-scrollbar {
  width: 12px;
  height: 12px;
}

.custom::-webkit-scrollbar-track {
  background: #f1f1f1;
  border-radius: 6px;
}

.custom::-webkit-scrollbar-thumb {
  background: #667eea;
  border-radius: 6px;
}

.custom::-webkit-scrollbar-thumb:hover {
  background: #5568d3;
}

/* Horizontal scrollbar */
.horizontal {
  overflow-x: auto;
  scrollbar-width: thin;
  scrollbar-color: #f59e0b #fef3c7;
}

/* Dark mode scrollbar */
@media (prefers-color-scheme: dark) {
  .container {
    scrollbar-color: #4b5563 #1f2937;
  }
}
