Native modals and dialogs, no JavaScript needed for basics

Modal Dialog
Native browser modal with backdrop blur and focus trap
Non-Modal Dialog
Can interact with page behind
 
        
/* Modal dialog (blocks interaction) */
.modal::backdrop {
  background: rgba(0, 0, 0, 0.5);
  backdrop-filter: blur(4px);
}

.modal {
  border: none;
  border-radius: 0.5rem;
  padding: 2rem;
  box-shadow: 0 20px 25px rgba(0,0,0,0.1);
}

/* Animating open/close */
dialog {
  opacity: 0;
  transform: translateY(-20px);
  transition: 
    opacity 0.3s,
    transform 0.3s,
    overlay 0.3s allow-discrete,
    display 0.3s allow-discrete;
}

dialog[open] {
  opacity: 1;
  transform: translateY(0);
}

/* Backdrop animation */
dialog::backdrop {
  opacity: 0;
  transition: opacity 0.3s;
}

dialog[open]::backdrop {
  opacity: 1;
}

/* Non-modal styling */
dialog.panel {
  position: fixed;
  right: 1rem;
  top: 1rem;
}
