The Past

Agent B: brief note

Old history records, the CSS galaxy was filled of cross browser anomalies, unidentified layout objects, clearfix wipes, prefixes and sprites. We kept the web stable, but it was a suffering mess.

Before modern CSS, we had to rely on: weird hacks, pork... ehm... workarounds, techniques that seem absurd today.

Let's take a nostalgic, and painful, trip down memory lane.
